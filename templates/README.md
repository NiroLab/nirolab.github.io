# NIRO Lab Website - Content Templates

Use these templates to submit your content for the NIRO Lab website.

## Templates Included

| File | Purpose |
|------|---------|
| `person-template.md` | Create your profile |
| `project-template.md` | Add a research project |
| `news-template.md` | Submit news or announcements |
| `gallery-template.md` | Submit photos for the gallery |
| `publication-template.md` | Add publications |

## How to Use

1. Download the template you need
2. Open it in any text editor (Notepad, VS Code, TextEdit, etc.)
3. Fill in your information following the instructions in each template
4. Delete the "INSTRUCTIONS" section before submitting
5. Save the file with the correct naming format (see each template)
6. Prepare your images according to the requirements

## Submission

1. **Create a ZIP file** containing:
   - Your completed `.md` file(s)
   - Your image file(s)

2. **Email to:** niro.laboratory@gmail.com

3. **Subject line format:**
   `NIRO Website - [Your Name] - [Content Type]`

   Examples:
   - `NIRO Website - John Doe - Profile`
   - `NIRO Website - Jane Smith - Project & News`
